# Agent Guardrails

This folder is designed as an agent-readable hook for future AqariX work. It converts the vibe-coding mistakes research into practical instructions agents should read before coding, designing, deploying, or refactoring.

## Files

- [AGENTS.md](./AGENTS.md): local agent instructions.
- [vibe-coding-mistakes-summary.md](./vibe-coding-mistakes-summary.md): condensed mistakes and prevention rules.
- [preflight-checklist.md](./preflight-checklist.md): checklist to run before starting a change.
- [release-checklist.md](./release-checklist.md): checklist to run before deployment or handoff.
