# Operations

## Operating Model

AqariX is not only software; it is a supervised marketplace. Operations must support trust, lead-room quality, seller/dealer success, agency delivery, and model calibration.

## Managed Lead Room Operations

Admin responsibilities:

- Monitor high-risk lead rooms.
- Review spam, abuse, and suspicious pricing flags.
- Resolve disputes.
- Support qualification workflow.
- Review off-platform leakage signals.
- Capture final outcomes for learning.

Operational metrics:

- New inquiries.
- Qualified lead rate.
- Viewing scheduled rate.
- Offer rate.
- Close rate.
- Average time in stage.
- Lost reason.
- Off-platform leakage rate.

## Seller/Dealer Success

Support responsibilities:

- Help sellers complete listings.
- Encourage better photos and copy.
- Explain pricing confidence.
- Help dealers use CRM pipeline.
- Encourage follow-up discipline.
- Convert trials to paid plans.

Metrics:

- Trial activation.
- Listings per seller/dealer.
- Assistant usage.
- Upgrade conversion.
- Lead response time.
- Dealer CRM retention.

## Agency Operations

Packages:

- Listing Starter.
- Reels and Social Push.
- Dealer Growth.
- Project Launch.

Workflow:

1. Seller/dealer selects package.
2. Agency operator receives task board.
3. Operator checks property details, audience, media requirements, delivery date, and approval status.
4. Assets are produced and submitted.
5. Seller/dealer approves or requests revision.
6. Approved assets attach to listing, Smart Boost, or dealer profile.
7. Performance is reviewed.

Metrics:

- Orders per package.
- Delivery time.
- Revision rate.
- Attach rate.
- Views, saves, lead-room starts, and cost per qualified lead.

## Support Operations

Support categories:

- Account access.
- Listing correction.
- Valuation dispute.
- Lead-room issue.
- Dealer CRM question.
- Agency order issue.
- Payment/subscription issue when added.
- Abuse or safety report.

Each support case should capture:

- User role.
- Affected entity.
- Severity.
- Status.
- Resolution.
- Whether docs, UX, or model behavior needs improvement.

## Weekly Operating Review

Review:

- Marketplace KPIs.
- Lead-room funnel.
- Listing quality.
- Dealer CRM usage.
- Agency queue.
- AI valuation disputes.
- Error monitoring.
- Cost/spend alerts.
- Security or abuse flags.

## Operations Guardrail

If the team cannot supervise, support, and recover a workflow, the workflow should not be treated as production-ready.
