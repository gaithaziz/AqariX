# Roadmap

## Phase 0 - Foundation

Timeline: Weeks 1-4

Outcomes:

- Data schema.
- PostGIS setup.
- Behavior event model.
- Listing feedback and ad-improvement loop.
- Listing ingestion plan.
- Initial zones.
- Auth roles.
- Design system.
- Documentation baseline.

## Phase 1 - AI/Data Core

Timeline: Weeks 5-12

Outcomes:

- AVM baseline.
- Comparable logic.
- Recommendation baseline using intake and behavior events.
- Forecast prototype.
- Offering-analysis API.
- Evidence and confidence object.

## Phase 2 - Role-Complete MVP

Timeline: Weeks 9-20

Outcomes:

- Buyer/investor app.
- Seller/dealer portal.
- Managed lead room.
- Admin dashboard.
- Agency order flow.
- Role-specific dashboards.

## Phase 3 - Closed Beta

Timeline: Weeks 21-28

Outcomes:

- 200-500 beta users.
- Seller/dealer trial.
- Supervised lead-room testing.
- Agency pilot.
- Model calibration.
- User feedback loop.

## Phase 4 - Public Launch

Timeline: Weeks 29-36

Outcomes:

- Amman/Irbid launch.
- Paid campaigns.
- Dealer CRM subscriptions.
- Smart Boosts.
- Premium reports.
- B2B outreach.

## Post-MVP Expansion

- Verified Listing Badge.
- Mortgage and affordability calculator.
- AqariX Score.
- Price-drop and opportunity alerts.
- Rental yield calculator.
- Off-plan project tracker.
- Diaspora mode.
- API access for institutions.

## Roadmap Guardrail

Launch all major roles early, but keep depth focused on the highest-value loops:

- Matched offerings.
- Offering analysis.
- Seller pricing.
- Managed lead rooms.
- Dealer CRM.
- Agency add-ons.
